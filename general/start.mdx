# Quick Start Guide

Get Wish Logger up and running in your Discord server with this step-by-step guide. No technical knowledge required\!

## Step 1: Invite Wish Logger

<Card title="Add to Discord" icon="circle-plus" color="#5865f2" horizontal href="https://discord.com/oauth2/authorize?client_id=1308500301791105044&permissions=8&scope=bot">
  Click here to invite Wish Logger to your server
</Card>

- **Click the invite link** above or visit our [website](https://wishlogger.xyz)
- **Select your server** from the dropdown menu
- **Review permissions** — Wish Logger needs these to function properly:
  - Ban Members
  - Kick Members
  - Moderate Members
  - Read Message History
  - Manage Messages
  - View Channel
  - Send Messages
  - Manage Webhooks
  - Embed Links
  - Manage Guild
  - View Audit Log
  - Manage Emojis
  - Manage Channels
  - Manage Roles

  ...or just **Administrator**
- **Click "Authorize"** to add the bot to your server

<Tip>
  **Tip:** Make sure you have "Manage Server" or "Administrator" permissions on the Discord server you want to add Wish Logger to.
</Tip>

---

## Step 2: Access the Dashboard

Once Wish Logger is in your server, set up logging channels through our web dashboard:

<Card title="Open Dashboard" icon="book-open" color="#00d9ff" horizontal href="https://wishlogger.xyz/dashboard">
  Configure Wish Logger from your browser
</Card>

1. **Visit the dashboard** at [wishlogger.xyz/dashboard](https://wishlogger.xyz/dashboard)
2. **Login with Discord** — Click "Login with Discord" and authorize the dashboard
3. **Select your server** from the list of available servers

---

## Step 3: Configure Logging Channels

Choose where you want your logs to appear:

### ~~Option A: Automatic Setup (Recommended)~~

1. **~~Click "Auto Setup"~~**~~ in the dashboard~~
2. **~~Let Wish Logger create channels~~**~~ — We'll automatically create organized logging channels:~~
   - `#message-logs`
   - `#member-logs`
   - `#voice-logs`
   - `#server-logs`
3. **~~Review and confirm~~**~~ the channel setup~~

### Option B: Manual Setup

1. **Create your own channels** in Discord (or use existing ones)
2. **In the dashboard**, assign each log type to your preferred channels:
   - **Mod Actions Logs** → Choose channel for moderator actions like mutes or kicks
   - **Member Logs** → Choose channel for joins/leaves/role changes
   - **Messages Logs** → Choose channel for message activity
   - **Voice Logs** → Choose channel for voice changes
   - **Actions Logs **→ Choose channel for various server action events
   - **Server** **Logs **→ Choose channel for server-level changes and updates
   - **Role Logs **→ Choose channel for role actions
   - **Channel Logs** → Choose channel for channel actions
3. **Save your settings**

<Warning>
  **Important:** Make sure Wish Logger has permission to send messages in your chosen logging channels.
</Warning>

---

## Step 4: Test Your Setup

Verify everything is working correctly:

1. **Edit or delete a message** in any channel
2. **Check your message logs channel** — You should see the logged event
3. **Try other actions** like joining a voice channel or changing your nickname
4. **Confirm logs appear** in their respective channels

<Success>
  **Success!** If you see logs appearing, Wish Logger is working perfectly. You're all set!
</Success>

## Common Questions

<Accordion>
  <AccordionItem title="Why aren't my logs showing up?">
  **Check these common issues:**

  * Wish Logger has permission to send messages in your log channels
  * You've assigned channels in the dashboard for each log type
  * The events you're testing are actually being logged (some events require specific permissions)
  * Try the `/test` command to verify the bot is responding
</AccordionItem>

  <AccordionItem title="Can I change my logging channels later?">
  **Yes!** You can change your logging channels anytime through the dashboard. Just:

  1. Go to [wishlogger.xyz/dashboard](https://wishlogger.xyz/dashboard)
  2. Select your server
  3. Choose new channels for any log type
  4. Save your changes
</AccordionItem>

  <AccordionItem title="Do I need to keep the dashboard open?">
  **No.** Once you've configured your settings, Wish Logger works automatically. The dashboard is only needed for initial setup and making changes.
</AccordionItem>

  <AccordionItem title="What if I want to stop logging certain events?">
  You can disable specific log types or set up filters through the dashboard. Visit the [Filter System](/docs/filters) documentation for detailed instructions.
</AccordionItem>
</Accordion>
