---
title: "List"
description: "Commands available in Wish"
---

| **Command**                    | **Description**                                                                                                                    |
| :----------------------------: | :--------------------------------------------------------------------------------------------------------------------------------: |
| /help                          | Get help and information about Wish.                                                                                               |
| /invite                        | Invite Wish to your server.                                                                                                        |
| /ping                          | Check the bot’s latency.                                                                                                           |
| /patreon                       | View Wish’s Patreon page.                                                                                                          |
| /terms                         | View Wish’s Terms of Service.                                                                                                      |
| /privacy                       | View Wish’s Privacy Policy.                                                                                                        |
| /ban                           | Ban a user from the server.                                                                                                        |
| /kick                          | Kick a user from the server.                                                                                                       |
| /unban                         | Unban a banned user.                                                                                                               |
| /unmute                        | Unmute a user.                                                                                                                     |
| /mute                          | Timeout (mute) a user.                                                                                                             |
| /clear                         | Clear a specified number of messages.                                                                                              |
| /reason                        | Update the reason for a specific moderation action. _(This only changes the reason shown in the log, not in Discord's audit log.)_ |
| /user                          | View a user's profile.                                                                                                             |
| /config (subcommand)           | View or reset the server configuration. _(Only the server owner can reset the config.)_                                            |
| /debug                         | View debugging information for your server.                                                                                        |
| /history (subcommand) (option) | View a user's history of bans, mutes, kicks, or nickname changes.                                                                  |
| /language                      | Change the bot's language for your server.                                                                                         |
| /logging (subcommand) (option) | Ignore specific users, channels, roles, or message content; or configure event groups for specific logging channels.               |
| /warn (subcommand) (option)    | Manage warnings: add, delete, change reason, transfer to another user, view user or server-wide warnings, or export warnings.      |
| /docs                          | Get a link to this documentation website.                                                                                          |
